BlinkUp Plugin Sample

This sample app shows off how to integrate the BlinkUp process with a Cordova/Phonegap application. It makes use of the BlinkUp Cordova plugin to interface with the BlinkUp SDK.

CAUTION

BlinkUp uses flashing light patterns. Please note that a very small percentage of individuals may experience epileptic seizures when exposed to such light patterns or flashing lights. If you have an epileptic condition, consult your physician prior to using this app to BlinkUp a device.
